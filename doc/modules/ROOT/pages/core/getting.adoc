= Getting Jandex

== | Maven Project

* your "pom.xml",
    ** add
+
[source,xml]
----
<dependency>
    <groupId>io.smallrye</groupId>
    <artifactId>jandex</artifactId>
    <version>${version.jandex}</version>
</dependency>
----

== Downloading Jandex

* download the link:https://search.maven.org/search?q=g:io.smallrye%20a:jandex[Jandex JAR]
